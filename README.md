# 📊 Sales Performance Dashboard

> Transforming raw sales data into actionable business insights through data analytics and visualization.

## Project Overview

The **Sales Performance Dashboard** is an interactive Power BI project designed to analyze and visualize sales performance across different regions, categories, and time periods.

This project aims to help stakeholders monitor key business metrics, identify trends, evaluate product performance, and make data-driven decisions.

---

## Project Objectives

✔️ Clean and prepare raw sales data

✔️ Analyze monthly, quarterly, and yearly sales trends

✔️ Identify top-selling and low-performing products

✔️ Compare sales across regions and categories

✔️ Calculate key business KPIs

✔️ Build an interactive Power BI dashboard

---

##  Key Performance Indicators (KPIs)

| KPI              | Description                |
| ---------------- | -------------------------- |
| 💰 Total Revenue | Total sales generated      |
| 📈 Total Profit  | Total profit earned        |
| 🛒 Total Orders  | Number of orders processed |
| 🚀 Growth Rate   | Sales growth over time     |
